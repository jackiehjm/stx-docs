--------------
Horizon access
--------------

This section describes Horizon access.

****************
Platform Horizon
****************

The platform Horizon provides access to StarlingX services for managing
physical infrastructure and host services.

The platform Horizon UI is available at
`http://<external OAM IP>:8080`.

*******************************
Containerized OpenStack Horizon
*******************************

The containerized OpenStack Horizon provides access to all OpenStack Services'
Horizon panels.

The containerized OpenStack Horizon UI is available at
`http://<external OAM IP>:31000`.
